CommandNPC (Fabric 1.21.1)

A lightweight Fabric mod that lets you create player-model NPCs which execute commands when right-clicked.

Features

Full player model NPCs (real skins)

Run any command on click

Per-player cooldown system

LuckPerms support

Custom colored & formatted name labels

Skin saved at time of assignment (does not change if player changes skin)

Commands
/cnpc create <id> <name>
/cnpc setcmd <id> <command>
/cnpc setcooldown <id> <seconds>
/cnpc skin <id> <player>
/cnpc skinhash <id> <hash>
/cnpc setlabel <id> <formatted text>
/cnpc remove <id>

Supports {player} placeholder in commands.

Requirements

Minecraft 1.21.1

Fabric Loader

Fabric AP


Compatible with luck Perms and Fabric API Permisions
